Git 操作步骤：

1.克隆仓库到本地：git clone https://github.com/bonnieyan/Test

2.切换到克隆下的目录：cd Test

3.创建文件：echo "# Test" >> README.md

4.提交到缓存区：git add .

5.提交备注：git commit -m “提交”

6.推送到远程仓库：git push -u origin master

7.后续提交文件重复3-5

8.后面推送到远程仓库只需要：git push

9.查看当前分支：git branch -a

Liunx 相关操作：

1.修改文件权限：chmod 777 文件夹

2.修改文件分组：chown yanjun:yanjun 文件夹

3.将所有文件的下层文件修改分组：chown -R yanjun:yanjun 文件夹
